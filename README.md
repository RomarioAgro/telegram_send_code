# telegram_send_code
будем писать телеграм бота который рассылает из магазинов коды авторизации для пробития чеков коррекции, да и вообще будут любые коды авторизации из розницы СБИС.
пример json:
{
    "shop": "ES",
    "text": "хочет пробить повтор чека",
    "number": "123456/01",
    "summ": "591.00"
}
